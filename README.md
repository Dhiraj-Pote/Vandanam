# Mangalacarana - Spiritual Library

A minimal, elegant spiritual website inspired by Vedabase.io design, built with Next.js 15 and Tailwind CSS.

## Features

- **Exact Vedabase.io UI replica** - Same colors, fonts, and layout
- **4 Spiritual Sections**:
  - MangalaCharan Prayers
  - Prayers to Recite Bhāgavatam  
  - Śikṣāṣṭakam (8 verses of instruction)
  - 10 Offenses While Chanting
- **Responsive Design** - Works on all devices
- **Static Export Ready** - Deploy to Netlify for free

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run development server:
```bash
npm run dev
```

3. Build for production:
```bash
npm run build
```

## Deployment

Deploy the `out/` folder to Netlify after running `npm run build`.

## Tech Stack

- Next.js 15 (App Router)
- Tailwind CSS
- TypeScript
- Merriweather font (Google Fonts)
